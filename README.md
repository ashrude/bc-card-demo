# BC Card Demo

A Basic demo to parse bc health/services/drivers licence card data.

It's built with SvelteKit and [rxing-wasm](https://github.com/rxing-core/rxing-wasm/tree/main).

## Usage

clone the repo, install the dependancies with `npm i`, and run with `npm run dev`

It can be hosted statically, as it is at https://bccarddemo.itsphere.ca
